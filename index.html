Trello App

參考了Trello的線上管理App，在此使用了Vue.js框架撰寫，利用VueCLI 3 搭建 並且使用Vuex 、Vue Router 管理資料與路由路徑。主要利用 HTML Drag and Drop API 實現卡片、任務拖曳效果(限PC端) 操作起來更直覺、也更方便，並可以更換主題顏色。

口罩地圖
Mask Map

因應2020年來的疫情爆發，造成全台灣口罩供不應求，利用這次事件來製作應用介面，來幫助需要的人。使用Vue.js框架撰寫，使用Axios串接 健保特約機構口罩剩餘數量明細清單 API。在地圖方面使用leaflet.js + OpenStreetMap製作 希望大家都可度過這次疫情
